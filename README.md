# Ecom Express Sales Dashboard | Power BI

## Overview
Interactive Power BI dashboard analyzing sales, revenue, product performance and cancellations.

## Features
- Revenue KPI
- Average Order Value
- Cancellation Rate
- Lost Revenue
- Total Orders
- Revenue by Product
- Revenue by Category
- Revenue by State
- Monthly Revenue Trend
- Product slicers
- Purchase date filter
